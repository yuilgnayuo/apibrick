
# Apibrick

A powerful **low-code API development platform** designed to simplify and accelerate API building, deployment, and management. Apibrick empowers developers and non-technical users alike to create robust APIs through intuitive visual configuration, eliminating the need for extensive hand-coding while supporting advanced customizations.

## Core Features
- **Visual Configuration**: Build and configure APIs via a drag-and-drop visual interface, reducing development complexity.
- **Dynamic Orchestration**: Support flexible dynamic orchestration of API workflows to meet complex business process requirements.
- **Encryption & Decryption**: Built-in data encryption and decryption capabilities to ensure the security of API data transmission and storage.
- **Multi-data Source Support**: Seamlessly connect to multiple heterogeneous data sources (relational databases, NoSQL, third-party services, etc.).
- **Groovy Script Engine**: Embed custom business logic with the integrated Groovy script engine for flexible API customization.
- **Multi-tenancy Architecture**: Native multi-tenancy support with isolated resources and permissions for different tenants/organizations.
- **Request/Response Handling**: Full lifecycle management of API requests and responses, including transformation, validation, and error handling.
- **Dual API Creation Modes**: Import and synchronize APIs automatically from Swagger/OpenAPI specifications, or create and edit APIs manually from scratch.
